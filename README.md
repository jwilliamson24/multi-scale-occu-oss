### multi-scale-occu-oss

## Salamander Occupancy Modeling

This project analyzes data from the Oregon Slender Salamander project using a multi-scale occupancy model. 
It includes data from 2013-2024.

## Code

multiscale-prepost-merge: combines data from pre-fire and post-fire surveys into one dataframe per species (ENES and OSS).  

multi-scale-occu-4D-V2-current: working code for the multi scale model. includes code for formatting detection and covariate data.

msc-occu-marginal-plots: creates marginal plots from model output.  
  

##### To run model script, choose the species detection matrix to use in line 280
